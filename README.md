## How It Works

1. **Add Users**: Create user profiles with name, role, email, and department
2. **Track Performance**: Add performance entries for each user across 5 metrics (0-100 scale)
3. **AI Analysis**: The system automatically:
   - Calculates weighted scores based on metric importance
   - Analyzes trends (improving/declining performance)
   - Rewards consistency
   - Generates tier-based rankings
4. **View Analytics**: Click on any user card to see detailed performance charts and history

## Performance Metrics

- **Productivity** (25% weight): Core output and efficiency
- **Quality** (25% weight): Work excellence and attention to detail
- **Communication** (15% weight): Team interaction and clarity
- **Initiative** (15% weight): Proactive behavior and self-direction
- **Collaboration** (20% weight): Teamwork and cooperation

## Ranking Tiers

- **Elite** (90-100): Exceptional performance
- **Excellent** (80-89): Strong, reliable performance
- **Good** (70-79): Solid performance
- **Average** (60-69): Meeting expectations
- **Below Average** (50-59): Below expectations
- **Needs Improvement** (<50): Significant improvement required
